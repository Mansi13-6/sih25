  # 🌍 Jharkhand Tourism Web Application (SIH25)

## 📌 Project Overview

This project is a **Tourism Web Application for Jharkhand** developed as part of **Smart India Hackathon (SIH25)**.
The platform helps tourists explore destinations, plan itineraries, and access useful travel services in one place.

---

## 🚀 Features

* 🏞️ Explore tourist places in Jharkhand
* 📅 Event Calendar for festivals & activities
* 🤖 Chatbot for user assistance
* 🗺️ Maps & Live Tracking
* 🧭 Personalized Itinerary Planning
* 📊 Sentiment Feedback System
* 🎥 Vlog Generation feature
* 🔐 User Authentication (Login/Register)

## ⚙️ Installation & Setup

```

```

2️⃣ Navigate to project folder

```
cd sih25
```

3️⃣ Install dependencies

```
npm install
```

4️⃣ Run the project

```
npm run dev
```
